# Task Scheduler

### 1.	ทำการ เข้าไปที่ Task Scheduler และ Create Task Scheduler
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/12.png)
### 2.	General จะเป็นการตั้งชื่อและใส่ Description 
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/13.png)
### 3.	ทำการสร้าง การกำหนดเวลาในการ Run Batch file
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/14.png)
### 4.	ทำการกำหนด วัน-เวลา ในการ Run Batch file
### ขั้นตอนที่ 1 กำหนดว่าจะ Run Program ตอนไหน เช่น รันครั้งเดียวจบ , รันทุกวันวันละครั้ง เป็นต้น
### ขั้นตอนที่ 2 ทำการเลือกวันที่ต้องการจะเริ่มทำงาน และเวลาที่ต้องการจะเริ่มทำงาน
### ขั้นตอนที่ 3 เป็นการกำหนด Run Program ตามที่ต้องการ เช่น กำหนดให้ run ทุก 1 ชม. 
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/15.png)
### 5.	สร้าง Actions ในการทำงานของ Program
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/16.png)
### 6.	ทำการกด Browser เพื่อเลือก Batch file ที่ต้องการ
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/17.png)
### 7.	ทำการเลือก Batch file ที่ต้องการ จากนั้นกด OK
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/18.png)
### 8.	Conditions คือ การกำหนดการทำงานของเครื่อง เช่น ตัวอย่างด้านล่าง ให้ทำงานเมื่อเครื่องทำงานเท่านั้น 
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/19.png)
### 9.	Settings คือ การกำหนดการเรียกใช้งาน Task Scheduler
![Editor preferences pane](https://github.com/youthza/BackupWindowsLinux/blob/main/Task_Scheduler/20.png)
### 10.	เมื่อทำการ ตั้งค่า Take Scheduler เสร็จสิ้น จะปรากฏหน้าต่างแบบนี้ขึ้นมาเพื่อแจ้งเตือนว่าจะทำงานยังไง
